# Cơm 24h - Ứng dụng đặt món trực tuyến

## Giới thiệu
Chào mừng đến với Cơm 24h - Hệ thống đặt cơm văn phòng, mì và nước giải khát giao hàng tận nơi 24/7.

## Hướng dẫn chạy
Trang web được xây dựng bằng HTML/CSS/JS thuần túy, không cần cài đặt server phức tạp.
Bạn có thể mở trực tiếp file `index.html` bằng trình duyệt.

**Đường dẫn file:**
`d:\web bán cơm\index.html`

## Tính Năng Đã Hoàn Thiện
1.  **Menu đa dạng**: 5 món cơm, 5 món mì, 5 món nước.
2.  **Tìm kiếm & Lọc**:
    *   Tìm theo tên món.
    *   Lọc theo danh mục (Cơm, Mì, Nước, Bán chạy).
    *   Sắp xếp theo giá (Tăng/Giảm).
3.  **Giỏ hàng**:
    *   Thêm món vào giỏ.
    *   Tăng/Giảm số lượng.
    *   Xóa món.
    *   Tự động tính tổng tiền.
4.  **Đặt hàng (Checkout)**:
    *   Form nhập thông tin (Tên, SĐT, Địa chỉ).
    *   Ghi chú món ăn.
    *   Chọn phương thức thanh toán (COD / Chuyển khoản).
5.  **Chi tiết món ăn**:
    *   Xem hình ảnh lớn.
    *   Mô tả chi tiết.
    *   Đánh giá sao.
6.  **Giao diện**:
    *   Thiết kế hiện đại, màu sắc kích thích vị giác.
    *   Responsive (tương thích mobile/tablet).

## Cấu trúc thư mục
- `index.html`: Giao diện chính.
- `styles.css`: CSS trang trí (Premium Style).
- `app.js`: Xử lý logic (Giỏ hàng, Lọc, Modal).
- `images/`: Chứa hình ảnh món ăn (SVG).

## Ghi chú
- Do không có backend thực tế, các chức năng Đặt hàng và Đăng nhập chỉ là mô phỏng (Mockup).
- Dữ liệu đơn hàng sẽ reset khi tải lại trang.
